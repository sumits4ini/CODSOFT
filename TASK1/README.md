# 🤖 Rule-Based Chatbot

A simple Rule-Based Chatbot developed using Python that responds to user queries based on predefined rules and pattern matching techniques. This project was created as part of the **CODSOFT AI Internship Task 1** to understand the fundamentals of conversational AI and Natural Language Processing (NLP).

---

## 📖 Project Overview

The Rule-Based Chatbot is designed to simulate basic human conversation by recognizing specific keywords and phrases in user input. Based on the detected pattern, it generates relevant responses using conditional logic.

Unlike advanced AI chatbots that use Machine Learning models, this chatbot relies on predefined rules, making it lightweight, easy to understand, and an excellent starting point for learning chatbot development.

## 🧠 Working Principle

The chatbot follows a simple workflow:

```text
User Input
     │
     ▼
Convert to Lowercase
     │
     ▼
Pattern Matching
     │
     ▼
Check Conditions (if-else)
     │
     ▼
Generate Response
     │
     ▼
Display Output
```

### Example

```text
User: Hello

Bot: Hi there! Nice to meet you.
```

```text
User: What is the time?

Bot: Current time is 10:45:32
```
